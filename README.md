# Currículo Profissional – LaTeX

Este repositório contém meu currículo profissional mantido em LaTeX, com múltiplas versões geradas a partir de uma única base de dados.

## Estrutura do projeto

- `dados.tex`  
  Fonte única de conteúdo (experiências, competências, formação).

- `cv-ats.tex`  
  Versão otimizada para sistemas de rastreamento de candidatos (ATS).

- `cv-gerencial.tex`  
  Versão focada em liderança, gestão de times e processos.

- `cv-executivo.tex`  
  Versão voltada a posições estratégicas e executivas.

## Princípios do projeto

- Separação total entre conteúdo e layout
- Versionamento para diferentes públicos
- Foco em legibilidade, ATS e clareza narrativa
- Manutenção centralizada do conteúdo

## Como usar

1. Edite apenas o arquivo `dados.tex` para atualizar informações.
2. Escolha o arquivo `.tex` desejado como principal no Overleaf.
3. Compile e gere o PDF correspondente.

## Ferramentas

- LaTeX
- Overleaf
- GitHub para versionamento e histórico
